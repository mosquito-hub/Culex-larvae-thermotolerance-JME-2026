Generalized Linear Models

Data preparation

All_data_longer <- All_data %>% pivot_longer(!c(Population, ID, Individuals), 
                                             names_to = "Nutrients", 
                                             values_to = "Concentration") %>% 
                  mutate(
  Food = case_when(
    grepl("FT1", Population) ~ "MD")) %>%
  mutate( 
    Region_of_Origin = case_when(
      grepl("FL", Population) ~ "FL",
      grepl("ME", Population) ~ "ME",
      grepl("AG", Population) ~ "AG",
      grepl("BG", Population) ~ "BG",
      grepl("TX", Population) ~ "TX"))

Global Model selection

First, comparison between additive and interactive models for an analysis that includes all the data.

model_no_interaction <- glm(Concentration~Region_of_Origin+Nutrients, data= All_data_longer)
model_interactions <- glm(Concentration~Region_of_Origin*Nutrients, data= All_data_longer)

anova(model_no_interaction, model_interactions, test="LRT")

Summary: There is a significant effect of considering the interaction between the food type and the region of origin. The additive model’s AIC is 1438, the interactive model’s AIC is 1396. Interactions between variables thus explains better the variability in the data.

Note: Pairwise comparisons can be run to inform on specific contrasts. For readability, the code below subsets only the pairwise comparisons that are significant.   mutate(prop_nut = Concentration/sum_nut) %>%
  mutate(sum_check = sum(prop_nut)) -> All_data_stacks
All_data_stacks$Individuals <- as.factor(All_data_stacks$Individuals)

ggplot(All_data_stacks, aes(x = ID, y= prop_nut, group = Population, fill = Nutrients)) +
